Nº Término Definición
1 HTML Lenguaje de marcado utilizado para estructurar contenido en la web.
2 CSS Lenguaje que permite aplicar estilos visuales a los elementos HTML.
3 JavaScript Lenguaje de programación que permite añadir interactividad a las páginas web.
4 HTTP Protocolo de comunicación utilizado en la web para la transferencia de datos.
5 HTTPS Versión segura del protocolo HTTP que utiliza cifrado SSL/TLS.
6 GET Método HTTP utilizado para solicitar datos de un servidor.
7 POST Método HTTP utilizado para enviar datos a un servidor.
8 Servidor Equipo o programa que proporciona servicios o recursos a otros dispositivos en red.
9 Cliente Dispositivo o programa que solicita servicios o recursos a un servidor.
10 Dominio Nombre legible que identifica a un sitio web en Internet.
11 Host Nombre asignado a un dispositivo conectado a una red.
12 URL Dirección que especifica la ubicación de un recurso en la web.
13 IP Dirección numérica que identifica a un dispositivo en una red.
14 DNS Sistema que traduce nombres de dominio a direcciones IP.
15 FTP Protocolo para la transferencia de archivos entre computadoras en red.
16 Backend Parte del desarrollo web que gestiona la lógica, base de datos y servidor.
17 Frontend Parte del desarrollo web que se encarga de la interfaz de usuario.
18 PHP Lenguaje de programación del lado del servidor para el desarrollo web dinámico.
19 MySQL Sistema de gestión de bases de datos relacional utilizado comúnmente en aplicaciones web.
20 isset Función en PHP que determina si una variable está definida y no es NULL.
21 Framework Conjunto de herramientas y buenas prácticas para el desarrollo eficiente de software.
22 API Interfaz que permite la comunicación entre diferentes programas o servicios.
23 JSON Formato ligero de intercambio de datos, fácil de leer y escribir.
24 AJAX Técnica para actualizar partes de una página web sin recargarla completamente.
25 Responsive Diseño que adapta una página web a diferentes tamaños de pantalla.
26 Bootstrap Framework CSS para crear sitios web responsivos y móviles.
27 SEO Optimización de sitios web para mejorar su posicionamiento en buscadores.
28 Cookie Archivo que almacena datos del usuario en su navegador.
29 Session Forma de almacenar datos del usuario mientras navega por un sitio web.
30 CMS Sistema para crear y administrar contenidos en una página web (ej. WordPress).
31 Git Sistema de control de versiones para el seguimiento de cambios en el código fuente.
32 GitHub Plataforma para alojar y colaborar en proyectos que usan Git.
33 Commit Acción de guardar cambios en un repositorio de control de versiones.
34 Branch Línea paralela de desarrollo en un repositorio Git.
35 Merge Proceso de combinar dos ramas de código en una sola.
36 SSL Protocolo de seguridad que cifra los datos entre cliente y servidor.
37 CMS Sistema de gestión de contenido que facilita la creación de sitios web.
38 Wireframe Esquema visual que representa la estructura básica de una página web.
39 SASS Extensión de CSS que permite usar variables, anidamiento y otras funcionalidades.
40 Viewport Área visible de una página web en la pantalla del usuario.
41 Meta tag Elemento HTML que proporciona información sobre la página web a los motores de búsqueda.
42 Form Elemento HTML que permite la entrada de datos por parte del usuario.
43 Input Elemento de formulario que permite al usuario ingresar información.
44 Textarea Elemento HTML que permite introducir texto en varias líneas.
45 Select Elemento de formulario HTML que permite elegir una opción de una lista desplegable.
46 Label Etiqueta HTML que describe un campo de formulario.
47 Div Contenedor genérico en HTML utilizado para estructurar contenido.
48 Class Atributo HTML usado para aplicar estilos a uno o más elementos.
49 ID Atributo HTML que identifica de forma única a un elemento en una página.
50 Viewport Área visible de la página web en un dispositivo, clave para el diseño responsivo.
